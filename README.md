# cracks generator

Генератор трещин-мозаик для SARAEJO street art festival.

## Страницы
- `index.html` — редирект на `cracks.html`
- `cracks.html` — основной инструмент: создание трещин, настройки, режим глубины, экспорт в JPEG
- `editor.html` — редактор элементов мозаики (кисти, формы, сохранение в архив)
- `preview.html` — режим бесконечного превью с автогенерацией трещин

## Шрифты
Папка `fonts/` должна оставаться рядом с HTML-файлами — шрифты подключены относительными путями (`@font-face`), а не встроены в код.

## Локальный просмотр
Просто открыть `index.html` (или `cracks.html`) в браузере — никакой сборки или сервера не требуется.
